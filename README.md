# Sensor-FW
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
## Supplies Needed
1. Arduino Uno
2. Ultra Sonic Ping Sensors
3. PIR Motion Sensors
4. Power Board

## Connection
1. PIR Motion Sensor Pin GND to Arduino GND
2. PIR Motion Sensor Pin VCC to Arduino 5V
3. PIR Motion Sensor Pin Out to Arduino D3

4. USP Sensor Pin GND to Arduino GND
5. USP Sensor Pin VCC to Power Board 5V
6. USP Sensor Pin Echo to Arduino D8
7. USP Sensor Pin Trig to Arduino D7
